결론

21막 — "검증된 방법"은 이미 시장에 있었다

v1~v3: LLM이 알려준 교과서 팩터(value·momentum·quality·lowvol). S&P500 대형주에서 측정한 예측력(IC-IR). 거래 가능 기준(0.30)에 한참 못 미침 = 노이즈.

팩터 IC-IR — 전부 0 근처. 가장 나은 momentum(+0.045)도 거래 가능 임계의 1/7. 검증 파이프라인(자작 PIT 지수 vs 실제 SPY corr 0.998)은 완성했지만, 이 후보군에 찾을 알파가 없었다.

3절대합 — 엔진(TQQQ) 대비 무엇을 얻고 무엇을 버렸나

못 이기면 낙폭이라도. 각 전략이 TQQQ 상방(로그수익)을 몇 % 따라갔고, TQQQ 고통(낙폭)을 몇 % 피했나. 이게 "TQQQ보다 매력 있나"의 직접 척도.

VT-CANON = 상방 74% 유지하며 고통 49% 회피. C1은 상방을 더 먹지만(85%) 고통 회피는 적음(41%). 어느 것도 "상방 다 먹고 고통 다 피하기"는 못 함 — 다시 공짜 점심 없음. 단 "버틸 수 있는 형태"로 바꾼 건 분명한 가치.

43막의 판결 — "타이밍"은 노출 효과였다 (A4 Stage A)

각 동적 정책을 "그날 TQQQ 비중 w_t" 기계로 환원 → 같은 평균 노출의 고정전략과 복리 비교. Timing Log Alpha = 양수면 진짜 타이밍.

5개 정책 전부 음수. 모든 동적 타이밍이 "같은 평균 비중 그냥 보유"보다 복리로 졌다(비용 전에도). "위험할 때 줄였다 늘렸다"의 Cash Drag가 방어 이득보다 컸다 — A4가 죽은 자리이자, 타이밍 알파가 없다는 직접 증거.

5내가 느낀 점 — 솔직하게

이 프로젝트에서 직접 겪고 깨달은 것.

1막 · 검증된 방법의 한계

초기 모델은 LLM(Claude·Codex)이 알려주는 검증되고 안전한 방법을 리서치해서 적용했다. v1에서 v7로 진화할수록 결국 그 안에서 분산·섬세함·안정성을 다듬는 작업이었지만 — 단순 SPY/QQQ 매수를 못 이겼다. 적어도 내가 검증한 공개데이터·일봉·비용후 후보군에서는, 이미 알려지고 검증된 안전한 방법으로 수익 내기가 어려웠다.

2막 · 알파는 어디서 오는가

차트 기술(모멘텀·엘리엇 파동·캔들)은 이미 알려져 시장에 반영돼 수익으로 안 이어진다는 걸 알았다. 알파는 공개된 차트 패턴이 아니라 남들이 안 파본 정보 우위(직접 리서치·전문가 의견 취합)와, 남들이 못 버티는 리스크·제약을 인수하는 데서 오는 것 같았다. 그래서 LLM이 비추천하는 길을 직접 찾고 레딧 등 전문가 의견을 모아 A1을 만들었다 — 근데 그것조차 단순 레버리지 TQQQ 매집을 못 이겼다.

3막 · 못 이기면 낙폭이라도

단순 TQQQ 매집을 수익으로 못 이긴다면, 낙폭이라도 완화해서 TQQQ보다 매력적인 시스템을 만들고 싶었다. 그래서 비율·수학 공식 — 타이밍 매도/매수를 잘 조절하면 되지 않을까에서 A4까지 왔다. 결과: 모든 타이밍이 같은 평균 노출 고정전략에 복리로 졌다. 수익은 타이밍이 아니라 TQQQ 노출 자체였다.

6결론

  • 내가 검증한 공개데이터·일봉·PIT·비용후 후보군에서는, 검증된·알려진·안전한 방법이 SPY/QQQ도 단순 TQQQ 노출도 못 이겼다 (8세대 robust negative).
  • 차트 기술·팩터·타이밍 = 알파 아님. 검증 인프라(자작 PIT 지수 vs 실제 SPY corr 0.998)는 완성했으나 이 후보군에 찾을 알파가 없었다.
  • 수익 엔진 = TQQQ 노출(레버드 베타)이지 알파가 아니다. 타이밍/Vault는 복리만 깎았다 (Cash Drag > Defense).
  • vol-target은 알파가 아니라 리스크 통제 — TQQQ 낙폭 -82%→-42%로 절반 (수익은 못 더하되 버틸 수 있게).
  • 가장 큰 자산 = 가짜 알파를 안 만든 검증 규율. look-ahead를 잡고, 사전등록 NULL을 지키고, 결과를 정직하게 negative로 적었다.
  • 길을 잃은 게 아니다 — 알파 프로젝트를 정직하게 끝냈고, 이제 '버틸 수 있는 레버 사이즈'를 고르는 단계다. 그게 정직한 종착점.
