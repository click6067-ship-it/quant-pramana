## 확정 파라미터
- **시장/유니버스:** KR(KOSPI200+KOSDAQ150 유동성필터) 먼저 → US 확장(2단계). **벤치 = KRX300 TR(primary) + 자체 cap-weight TR(secondary 대조).**
- **방향:** KR 롱온리(공매도=모델링 변수 — FSC 2025-03-31 재개·NSDS). 롱숏 = US 단계 별도 상태머신.
- **자본:** 가상 ₩100M, paper지만 **live 전환 가능 설계**. 정수주(integer-share).
- **빈도:** 일봉 스윙/중장기 cross-sectional.
- **게이트:** ① 수익노브 OPEN(멀티알파·종목/섹터캡↑·턴오버↑·Kelly는 검증게이트 후 tiny fractional) · ② 검증 킬게이트 + ③ 결정론 리스크veto·LLM off-path = LOCKED.
- **성공 바(provisional):** 비용·세금후 KRX300 TR 초과 ≥+3%p · 초과 IR≥0.5 · DSR>0@95% · MDD≤1.2×벤치 · 2×비용서 net-positive · OOS≥3y. **킬:** DSR≤0 또는 2×비용 초과 IR<0.3 → 중단·단일 베이스라인 회귀.

## 모델패밀리 판정 (Stage 0B 검증)
- **KEEP:** tree ensemble(LGBM/XGB/CatBoost — 베이스라인 알파) · 투자자유형 flow/NAT(KR 1순위 신호, 동료심사) · Ledoit-Wolf+HRP(risk) · linear/penalized(해석 베이스라인+벤치 팩터) · 결정론 veto.
- **ADAPT:** q5 expected-growth(KR 팩터 우선) · optimizer(변환만, alpha 아님) · regime(리스크게이트 한정) · LLM-MCTS(가설생성만).
- **DEFER:** conditional autoencoder(게이트 후·KR 자체재현 선결).
- **DISCARD:** TSFM 전부(off-the-shelf/from-scratch/Kronos — 비용후 GBDT 못 이김 + 누수 만연) · LLM on-path(보고 alpha≠배포증거) · 고레버리지 콘테스트 스타일(반면교사).
