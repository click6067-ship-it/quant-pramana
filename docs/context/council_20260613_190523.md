임팩트: A2-T +851%, dynamic -86%p, beat flags 전부 execution model 불확정.  
수정안: OHLC를 받아 `signal close t-1 → execute open t → open-to-close t`로 계산하거나, close-only면 보수적으로 신호를 한 바 더 늦춰 `mds[i-2]`가 `close_{i-1}→close_i`에 적용되게 하라.
