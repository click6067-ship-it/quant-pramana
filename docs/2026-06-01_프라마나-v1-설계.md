# 프라마나 v1 — 설계 (ACTIVE · 2026-06-01 확정)

> 풀파워 회의 산출(Claude 설계 + Codex 적대리뷰). 7-layer institutional(SHELVED) → ₩10M 80%-절삭판 → **Codex 정정** = 이게 빌드 타깃.
> **불변식(절대 보호):** LLM off-path · 결정론 게이트 on-path · *알파 정교함보다 데이터·비용 정직이 먼저.*

## 확정 파라미터
- **시장/유니버스:** KR / KOSPI / **python-kis 모의투자(paper)**
- **방향:** **롱온리**(KRX 공매도 제약 — 무차입 금지·차입검증)
- **자본:** 실제 **₩10M**에서 **정수주(integer-share) 시뮬**(코즈메틱 아님 — 라운딩·현금drag·미체결이 1차효과). capacity sweep = 나중 진단.
- **빈도:** 일봉/스윙, cross-sectional ranking

## 아키텍처 (7 레이어)
```
[거버넌스(사람): 한도·승인·kill]
[LLM sleeve(주문권 0): 가설·데이터요구 + 누수/과적합/리스크 공격자 → 체크리스트·테스트케이스만 (가설공장 금지)]
[결정론 게이트(코드): 최대포지션·종목/섹터비중·턴오버·일일손실·MDD정지·유동성플로어·상한/정지/stale 거래금지]

══════════ ↑ 전부의 토대(먼저 짓는다) ↓ ══════════
[Layer 0 — 데이터정합·거래가능성: PIT·생존편향제거·수정주가·상폐/정지/CA·±30%·T+2·1주·틱]
[검증랩: purged walk-forward·Deflated Sharpe·FWER/FDR·prereg]
```

## v1 스코프 (먼저 만드는 것)
**Layer 0(데이터정합) + 검증랩 + 결정론 게이트 + 베이스라인 1개.**
**Deferred(v1 제외):** TSFM, HRP, Ledoit-Wolf, CVaR, Kelly-사이징, 롱숏, 멀티알파/앙상블.

## Go / No-Go 킬 조건 (Codex)
> **지루한 베이스라인 1개(예: KOSPI200 동일가중 또는 cross-sectional 모멘텀)를 — 비용·세금 포함·정수주·생존편향/PIT 클린 데이터로 — 재현하지 못하면, 알파 빌드를 중단한다.**

## 가장 큰 리스크 (Codex, FACT/OPINION)
- **#1:** 비PIT/생존편향/CA-미조정 KR 데이터 위에 화려한 검증탑을 쌓고 그 결과를 믿는 것 = theater → 3주도 진짜 경로도 날아감. 그래서 Layer 0가 토대.
- 솔로 스케일에선 **운영리스크**(API·거부·CA·휴장·중복주문)가 공분산오차보다 빨리 죽임.

## 변경 이력 (₩10M v1 → 정정판)
- **+1 레이어:** Layer 0(데이터정합·거래가능성) 신규 — 검증랩에서 분리·승격.
- **Decision 축소:** HRP·Ledoit-Wolf·CVaR·TSFM·Kelly-사이징 → 전부 deferred. inverse-vol+하드캡만.
- **게이트:** CVaR/vol예산 → 단순 결정론 하드캡(짧은 일봉 히스토리에 CVaR 신뢰불가).
- **방향:** 롱숏 모호 → **롱온리** 명시.
- **Execution:** 정수주·경험적 체결·운영리스크 디테일 추가.
- 본질: *정교함을 예측기/최적화에서 빼서 토대(데이터·체결·검증·리스크)로 이동.* 구조는 +1, 만들 복잡도는 ↓.
